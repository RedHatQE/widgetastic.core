Comprehensive Edge Case Testing Table
Single table designed to test multiple edge cases and error conditions
Comprehensive Edge Case Testing
ID Key Name Name Input Status
1 duplicate First First-Copy Active
2 duplicate Second Second-Copy Inactive
3 unique Third Third-Copy Processed
4 another Fourth Fourth-Copy Pending
Comprehensive edge case testing table covers:
- Duplicate headers: "Name" appears twice (columns 2 and 3)
- Duplicate keys: "duplicate" value in Key column (rows 1 and 2)
- Widget elements: Text inputs and checkboxes in Input column (column 4)
- Row attributes: data-test and data-category for filter testing
- Unique statuses: Active, Inactive, Processed, Pending for normal testing
